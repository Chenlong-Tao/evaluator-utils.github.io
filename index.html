Chat Completions Create 参数编辑器

将 client.chat.completions.create(...) 的请求参数做成可视化表单。通过校验后才会生成可复制的压缩 JSON。

开始使用 查看说明

工具

全部在浏览器本地运行，不上传数据。

client.chat.completions.create

全部参数（openai==1.99.9，除 model/messages 外）
额外字段（非 SDK，可选，JSON object）

关于

这是一个纯静态的 Chat Completions Create 参数编辑器：HTML + CSS + Vanilla JS，适合放在 GitHub Pages。 你在表单里修改参数后，页面会进行类型/语法校验；只有校验通过才会生成可复制的压缩 JSON。

提示：为了避免 Jekyll 处理导致的路径/构建差异，本仓库默认带 .nojekyll。
